# RoadMemo V0.1 发布前检查清单

## 1. 构建与安装

- [ ] `assembleDebug` 可稳定通过
- [ ] 应用首次安装可正常启动
- [ ] 首次启动无崩溃、无白屏
- [ ] 应用名称、包名、版本号正确

## 2. 首次使用链路

- [ ] 无车辆时首页能正确提示添加车辆
- [ ] 新增第一辆车后自动成为默认车辆
- [ ] 新增第一条能源记录后首页摘要可刷新
- [ ] 新增第一条保养记录后记录页可刷新
- [ ] 新增第一条费用记录后统计页可刷新
- [ ] 新增第一条续期事项后提醒页可看到提醒

## 3. 记录维护链路

- [ ] 能源记录可新增 / 编辑 / 删除
- [ ] 保养记录可新增 / 编辑 / 删除
- [ ] 费用记录可新增 / 编辑 / 删除
- [ ] 续期事项可新增 / 编辑 / 删除
- [ ] 删除前都有确认提示
- [ ] 删除续期事项会联动删除提醒来源

## 4. 提醒链路

- [ ] 提醒页可看到当前默认车辆的提醒
- [ ] 提醒可标记为已处理
- [ ] 提醒可忽略
- [ ] 提醒可跳回来源记录
- [ ] 提醒处理后会从待处理列表移除

## 5. 首页与统计

- [ ] 首页能正确展示本月总支出
- [ ] 首页能展示最近能源 / 保养摘要
- [ ] 首页提醒入口可跳转到提醒页
- [ ] 统计页月度趋势可展示
- [ ] 统计页分类汇总可展示
- [ ] 统计页最近表现可展示

## 6. 数据能力

- [ ] CSV 导出可生成文件并调起系统分享
- [ ] JSON 备份可生成文件并调起系统分享
- [ ] 恢复备份前会弹确认框
- [ ] 恢复成功后数据可回到列表中
- [ ] 恢复失败时会显示错误信息

## 7. 视觉与交互一致性

- [ ] 首页、记录页、统计页、车辆页风格一致
- [ ] 四张表单页结构统一
- [ ] 设置页和提醒页风格已收口
- [ ] 主按钮、次按钮、错误提示表现一致
- [ ] 空状态文案清晰，不出现开发占位词

## 8. 发版前需要补齐的非功能项

- [ ] 应用图标与启动图资源
- [ ] 隐私说明 / 数据说明
- [ ] 备份恢复策略说明
- [ ] 通知权限策略说明
- [ ] 内测安装说明

## 9. 当前不阻塞 V0.1 的已知缺口

- [ ] 暂无正式云同步
- [ ] 暂无 OCR 和附件
- [ ] 暂无自动化测试覆盖
- [ ] 暂无正式后台提醒调度闭环
- [ ] 暂无多设备数据冲突处理

## 10. 冻结建议

满足以下条件后，可视为 `V0.1` 冻结候选：

- 主链路可连续体验
- 无明显崩溃或阻塞性错误
- 数据可导出、可备份、可恢复
- 页面视觉风格已基本统一
- 文档、README、构建方式清晰可追溯
